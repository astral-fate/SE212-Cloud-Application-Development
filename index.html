Course Objectives

  • ✓ Understand the fundamental shift from Traditional IT to Cloud-native architectures.
  • ✓ Master Docker container lifecycle: Image creation, storage management, and networking.
  • ✓ Deploy resilient applications using Kubernetes orchestration and deployment strategies.
  • ✓ Compare serverless computing options (Lambda/GCF) vs managed container services (ECS/EKS).
  • ✓ Differentiate between Virtual Machine virtualization and Container (OS-level) virtualization.
  • ✓ Utilize Infrastructure as Code principles with Docker Compose and K8s manifests.
